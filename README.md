# SCM Project - Calculator

## Description
A simple Java calculator project used to practice Git branching, merging, and resolving merge conflicts.

## Features
- Addition feature (implemented in `feature/addition` branch)
- Subtraction feature (implemented in `feature/subtraction` branch)

## Git Workflow
1. Created branches:
   - `feature/addition`
   - `feature/subtraction`
2. Implemented different features in each branch.
3. Merged branches into `main`.
4. Resolved a merge conflict in `Calculator.java` when both branches modified the same section.

## How to Run
- Open the project in VS Code.
- Run `Calculator.java` using Java extension or terminal.
